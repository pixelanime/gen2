Tentang CreatorKit AI

Diperbarui: Januari 2025

CreatorKit AI adalah platform alat kreatif dan download media berbasis web. Semua proses berjalan di browser tanpa mengirim data ke server kami.

Fitur

  • Generate Foto AI — Gambar procedural dari deskripsi teks.
  • Buat Banner — Banner untuk sosial media.
  • Foto ke Video — Video animasi 10 detik.
  • Buat CV — CV profesional siap lamar.
  • Prompt Generator — Prompt untuk AI gambar.
  • Stiker Lucu & Foto ke Stiker — Stiker kustom.
  • Download MP3 & Video — Dari YouTube, Facebook, TikTok via Cobalt API.

Kebijakan Privasi

Berlaku: Januari 2025

CreatorKit AI tidak mengumpulkan data pribadi. Semua proses berjalan di browser (client-side). Statistik kunjungan disimpan di localStorage perangkat dan tidak dikirim ke server manapun.

Cookie teknis minimal digunakan untuk menyimpan preferensi tema. Jika iklan AdSense ditampilkan, pihak Google mungkin menggunakan cookie sendiri. Atur di Google Ad Settings.

Semua file yang dibuat sepenuhnya diproses di perangkat Anda. Kami tidak memiliki akses terhadap file tersebut.
